ROMANTIC GREETING CARD WEBSITE

NEW FEATURES ADDED
==================

1. CINEMATIC REVEAL EFFECT
---------------------------
When the envelope overlay closes and slides away, the main content appears with a cinematic camera-focus effect:

- Initial state: slightly blurred, zoomed in (scale 1.08), transparent (0.7 opacity)
- Final state: sharp focus, normal size (scale 1), full opacity (1.0)
- Transition: smooth 1.2s ease-out animation

This creates a professional, movie-like reveal as the page content comes into focus.

CUSTOMIZATION:
- Edit the `.main-content.cinematic-hidden` class in style.css to adjust:
  * transform: scale() - zoom level
  * filter: blur() - blur intensity
  * opacity - transparency
- Edit the `.main-content` transition property to change animation duration


2. FLOATING HEARTS ANIMATION
-----------------------------
Subtle romantic hearts float upward in the background after the envelope is opened:

- 35 floating hearts by default
- Soft pink and light red color variations
- Hearts drift upward and fade out at the top
- Resets continuously for endless animation
- Uses Canvas API with requestAnimationFrame for smooth performance

CUSTOMIZATION:
In script.js, you can adjust:

HEART_COUNT constant (line ~320):
- Change the number of hearts (recommended: 30-40)
- Lower number = better performance
- Higher number = more romantic atmosphere

HeartParticle class constructor (line ~250):
- this.size - adjust heart size range (default: 6-20px)
- this.speedY - adjust upward speed (default: 0.5-1.5)
- this.speedX - adjust horizontal drift (default: -0.25 to 0.25)
- this.opacity - adjust transparency (default: 0.4-0.8)
- colors array - add or modify color variations

ANIMATION FLOW:
1. User clicks envelope
2. Envelope opens (0.8s)
3. Overlay slides away (0.8s)
4. Cinematic reveal begins (1.2s)
5. Floating hearts start
6. Typing animation starts

PERFORMANCE:
- Hearts are limited to prevent performance issues
- Canvas is automatically resized on window resize
- Animation uses requestAnimationFrame for optimal performance
